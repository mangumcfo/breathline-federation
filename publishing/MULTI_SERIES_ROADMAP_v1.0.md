## 6. Execution Rules

### Rule 1: Batch Outlines for a New Series in 2-3 Days

| Day | Activity | Output |
|-----|----------|--------|
| Day 1 (4 hrs) | Topic research, keyword analysis, competitor scan. Write 12 titles + subtitles. | Series skeleton with all titles |
| Day 2 (4 hrs) | Write detailed chapter outlines for Books 1-4 (full v1.2 format). Summarize Books 5-12. | Complete series plan |
| Day 3 (3 hrs) | Run editorial board review. Merge keyword optimization. Finalize v1.2. | Production-ready series plan |

### Rule 2: Quality Gates Before Adding to Backlog (v1.1 — includes Book 1-3 learnings)

A book enters the "READY" backlog only when ALL of these are true:

**Content Gates:**
- [ ] Manuscript complete (all chapters + prompt library + case studies)
- [ ] Editorial board review completed and fixes implemented
- [ ] Kenneth's human review completed (voice notes per chapter)
- [ ] Book 1 learnings applied (Industry Signals, Sources appendix, case study framing, frontier models, cross-refs)
- [ ] Word count ≥ 8,000 words / ≥ 55 pages

**Quality Scan Gates (automated):**
- [ ] Zero "GPT-4" references (`grep -c "GPT-4" manuscript` = 0)
- [ ] Zero first-person in case study chapter (`grep "\bI \b\|I built\|my team" case_study_chapter` = 0)
- [ ] 11 Industry Signal vignettes present (one per chapter, with validated URLs)
- [ ] Sources & Further Reading appendix present
- [ ] TOC chapter titles match actual headings (including `&amp;` entity matching)
- [ ] No duplicate paragraphs or tool landscape lines

**Build Gates:**
- [ ] All figures generated and embedded (matplotlib navy/gold, ≤1425px, 300 DPI)
- [ ] PDF builds with no errors (WeasyPrint, 6x9, 0.75" margins, no text overflow)
- [ ] EPUB builds and includes all content
- [ ] Cover generated (front PNG + JPG for Kindle + paperback wrap PDF + hardcover wrap PDF)
- [ ] Amazon description written (~150 words, HTML-formatted)
- [ ] 7 KDP keywords assigned
- [ ] Ideological scan clean

**This gate applies to ALL series, not just Agentic AI.** The standards from Book 1-3 production are the baseline for every book going forward.

### Rule 3: Metrics to Add a 3rd Active Series

Add Slot C to active publishing (3 series simultaneously) ONLY when:

- [ ] Monthly KU page reads exceed 30,000 (proving audience demand)
- [ ] Email list exceeds 500 subscribers (proving distribution capacity)
- [ ] Kenneth's weekly time commitment is under 20 hours (proving capacity)
- [ ] Backlog is at 6+ weeks across existing series (proving buffer)
- [ ] Revenue from books exceeds $2,000/month (proving economics)

If ANY of these are not met, stay at 2 active series. Quality > quantity.

### Rule 4: Never Publish Without a 2-Week Buffer

If the backlog drops below 2 weeks of ready books, STOP marketing activities and redirect all time to writing. A missed publish date is less damaging than a rushed, low-quality book.

### Rule 5: Seasonal Adjustments

| Period | Adjustment |
|--------|-----------|
| Q4 (Oct-Dec) | Increase publishing velocity — gift season + New Year resolution buyers |
| Q1 (Jan-Mar) | Launch new series — "fresh start" energy drives new topic adoption |
| Q2-Q3 (Apr-Sep) | Steady state — maintain cadence, build backlog for Q4 push |
| Any week with < 3 reviews per book | Pause new publishing, focus on ARC outreach and review generation |
